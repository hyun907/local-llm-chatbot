## 2순위 — 기능 확장

### 4. 대화 이력 영구 저장 (SQLite) ⏱ 반나절
- 현재 한계: 서버 재시작 시 전체 대화 소멸 (메모리 dict)
- 세션 ID가 이미 있으므로 `messages(session_id, role, content, created_at)`
  스키마로 자연스럽게 이관. 오래된 세션 만료 정리도 함께
- 완료 기준: 서버 재시작 후에도 같은 브라우저에서 문맥 유지

### 5. RAG 문서 업로드 UI ⏱ 반나절
- 현재 한계: `data/`에 txt를 직접 넣고 재시작해야 인덱싱
- 브라우저에서 txt 업로드 → 즉시 청크·임베딩 추가 (`POST /documents`)
- 업로드 문서 목록·삭제 UI까지 하면 RAG 데모 완성형

### 6. Docker화 ⏱ 반나절
- `ollama serve` + Flask를 docker-compose로 묶어 "클론 → 한 줄 실행"
- Apple Silicon에서 Docker 내 Ollama는 GPU(Metal) 미지원이라 느려짐 —
  README에 "네이티브 실행 권장, Docker는 이식성용" 명시 필요

### 7. 성능 비교 대시보드 ⏱ 자유
- `docs/benchmark_raw.json`을 그래프로 시각화 (모델별 tok/s, TTFT)
- README 표를 차트 이미지로 승격하는 수준이면 충분
